**Batch:** 2026  
**Organization:** DecodeLabs  
**Language:** Java  

---

## About The Project

A console-based Number Guessing Game built as part of the
DecodeLabs Industrial Training Program. The program generates
a random number between 1 and 100 and challenges the player
to guess it within 7 attempts using High/Low feedback.

---

## Features

- Random number generation using java.util.Random
- User input handling using java.util.Scanner
- High / Low feedback system
- Attempt limiter (7 attempts per round)
- Score system based on attempts used
- Multiple rounds with replay option
- Input validation with try-catch (no crashes)
- Clean method-based code structure
